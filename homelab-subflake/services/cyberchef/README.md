# CyberChef

The Cyber Swiss Army Knife - a web app for encryption, encoding, compression and data analysis.

## Overview

CyberChef is a web application for performing crypto, encoding, compression, and data analysis operations in your browser. This deployment serves the static application via nginx.

Access at: `https://cyberchef.srv.<publicDomain>`

## Documentation

- [Official CyberChef Wiki](https://github.com/gchq/CyberChef/wiki)
- [Recipe Collection](https://github.com/mattnotmax/cyberchef-recipes)

## Implementation Notes

- Stateless (no persistent storage or backups)
- Static files served from nix package
- No authentication required
